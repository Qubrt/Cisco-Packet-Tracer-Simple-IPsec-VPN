# Cisco-Packet-Tracer-Simple-IPsec-VPN
A simple configuration for IPsec VPN in network-simulating program Cisco Packet Tracer.<br>

## How to use
You need to have the program Cisco Packet Tracer.<br>
The version should be at least 8.2.2.0400<br>

Then simply open the file with Cisco Packet Tracer.<br>
