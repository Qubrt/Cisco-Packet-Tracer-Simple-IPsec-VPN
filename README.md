# Cisco-Packet-Tracer-Simple-IPsec-VPN
A simple configuration for IPsec VPN in network-simulating program Cisco Packet Tracer.<br>

## A new version has been added!
A new version is now avalable in the **office-branch**. It feateres a more realistic scenario with two offices, including a DHCP server, Wireless Access Point and Vlan integration.

## How to use
You need to have the program Cisco Packet Tracer.<br>
The version should be at least 8.2.2.0400<br>

Then simply open the file with Cisco Packet Tracer.<br>
Be wary that the documentation is in Bulgarian, however the configuration commands are in English.<br>
